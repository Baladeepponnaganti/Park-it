# Park-IT

Park-IT is a smart parking management application that enables users to find, book, and manage parking spaces efficiently. Property owners can list their available parking spaces, while users can search, reserve, and pay for parking through a seamless web interface.

## Features

- User Authentication
- Parking Space Listings
- Search & Filter
- Booking System
- Owner Dashboard
- Responsive UI

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Getting Started

```bash
npm install
npm start
```
